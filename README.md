## 🚀 Getting Started

To explore the code locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Samiran2004/Codes.git
    cd Codes
    ```

2.  **Navigate to a directory**:
    ```bash
    cd TUF/Array  # Example: Go to TUF Array problems
    ```

3.  **Run a solution**:
    - **Java**:
      ```bash
      javac Solution.java
      java Solution
      ```
    - **Python**:
      ```bash
      python3 script.py
      ```
    - **JavaScript/TypeScript**:
      ```bash
      node script.js
      # OR for TypeScript
      npx ts-node script.ts
      ```
